## How To Use

1. Copy the `.hh3` file into the **root directory** of your calculator  
   (do not place it inside a subfolder).

2. Launch the calculator and open the Hex Editor through Hollyhock Launcher.  

3. Controls:  
   - Cursor: `←` / `→`  
   - A, B, C, D, E, F: `=` / `3` / `𝒙` / `𝒚` / `𝒛` / `^` / `÷`
   - `Clear` = quit
